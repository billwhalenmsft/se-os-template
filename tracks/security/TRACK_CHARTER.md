# Track Charter — Security

> **OU Lead:** Michael Billy (CVP, Security)  
> **Americas SE&O:** Scott Prather

Security SEs cover the full Microsoft security portfolio — identity, endpoint, cloud, data protection, and SIEM/SOAR. Manufacturing is a high-priority vertical for Security due to OT/IT convergence, ransomware exposure, and increasing regulatory pressure.

---

## Outcome statement

> **Security SEs show customers that Microsoft Defender, Sentinel, Purview, and Entra form a unified security platform — not a collection of tools — enabling a SOC to detect, investigate, and respond faster while reducing vendor sprawl and license cost.**

---

| Motion | Key Products | Primary Demo Surface |

---

## Manufacturing-specific angle

Discrete manufacturing customers are a prime Security target:

- **OT/IT convergence** — plant floor systems (PLC, SCADA) increasingly connected to Azure
- **Ransomware exposure** — manufacturing is the #1 targeted industry
- **Defender for IoT** — purpose-built for OT environments, integrates with Azure + Sentinel
- **Regulatory pressure** — CMMC (Defense Industrial Base), NIST CSF, ISO 27001

For joint demos with Azure (IoT Security) or AI Business Process (identity for D365), coordinate via `../_shared/discrete-mfg/`.
