## How to use this folder

1. **Start here** before any customer call — read the sub-vertical's `README.md` for personas and talk tracks
2. **Then go to your OU folder** for the product-specific use cases and demo sequences
3. **For joint demos** (cross-OU), the shared patterns are the coordination layer

Open a GitHub issue with label `shared:<topic>` to propose new shared content.
