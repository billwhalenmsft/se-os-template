05 / 08
Capabilities
Built for how the team actually works
Every feature is designed for a real workflow pain point
🎯

Outcomes Editor

Create, edit, and track strategic outcomes with priority, status (Active / Achieved / Paused), emoji, and linked GitHub issues. Custom outcomes persist in your browser — nothing requires a backend.

📱

Quick Capture

Mobile-optimized idea capture — Idea, Use Case, SOP Request, or Bug. With a GitHub PAT it files issues directly via API. Without one, it opens a pre-filled GitHub form. Bookmark ?quick=1 for phone speed-dial.

💬

Agent Forum

Threaded discussion board backed by GitHub Issues. Agents post updates, Bill steers direction, all tracked in the repo. Topics can have comments, replies, and emoji reactions.

📋

Kanban Board

Live view of GitHub Issues filtered by mfg-coe label — Backlog, Active, Needs Bill, Done. Issue counts auto-refresh from GitHub API.

📦

Agent Library

Browse and install agents from RAR Registry, AIBAST, Community, Agent 365, or custom GitHub sources. Two-section layout: Installed vs. Available. Core CoE agents cannot be removed.

🏆

Achievement Cards

12 flip-card milestones with XP tracking. Unlocks based on real activity — installing agents, filing issues, adding sources, earning admin access. Gamifies CoE adoption.

🔵

Agent 365 Integration

Connect your Microsoft tenant to browse Copilot Studio, Azure AI Foundry, and Marketplace agents all in one place. Install/uninstall from the same interface as local agents.

🌙

7 Color Themes

Dark, Light, Neon, Forest, Midnight, Sunset, Ocean — all persisted to localStorage. Neon theme triggers a boot sequence animation. Easter egg: type "coe" anywhere.

Architecture
Lightweight by design
No frameworks, no build step, no dependencies to update

⬇
🔑 GitHub API — Issues · Comments · Labels · Repos (PAT-authenticated)
⬇
⚡ Azure Functions Backend — GPT-4o · Agent Routing · Azure File Storage
⬇
🏭 D365 CE — Master CE Mfg · Mfg Gold Template · Copilot Studio Agents
Key Design Decisions
✦ Single HTML file — deploy anywhere, no build pipeline needed
✦ GitHub Issues as the database — no separate backend for content
✦ localStorage for user prefs — zero auth complexity for personal settings
✦ Azure Static Web Apps — free tier, custom domain, auto-HTTPS

How agents are built
RAPP Pipeline
Rapid AI Agent Production Pipeline — transcript to deployed agent in minutes
1. Discovery Transcript
Drop a call recording or notes into the RAPP agent — it extracts use cases, personas, and requirements automatically
2. Agent Code Generation
RAPP generates Python agent class, metadata schema, test cases, and deployment config in one step
3. Quality Gates QG1–QG6
Six automated checkpoints: requirements, design, implementation, testing, demo, and deployment readiness
4. Deploy to Azure + Copilot Studio
Auto-pushes to Azure File Storage, available in Command Center chat immediately; YAML exported for Copilot Studio
⚡ Fast path: transcript_to_agent action goes from raw transcript → deployable Python agent + demo JSON + HTML tester in a single call
