❓ Anticipated Questions

▼
❓ How do we keep competitor lists current?
The agents have configurable competitor dictionaries. Adding new competitors is a simple configuration update - no code changes needed. We recommend quarterly reviews.
❓ What if we want to add a data source?
The agents are designed for API integration. Adding trade publications like PM Engineer, Supply House Times, or internal databases is straightforward integration work.
❓ How does this integrate with Power BI?
Every agent returns structured JSON that can feed directly into Power BI, your data warehouse, or any BI tool. CSV exports are also available.
❓ Real-time alerts vs. quarterly reports?
The system supports both. Quarterly is the batch mode shown here. The same agents can be triggered on-demand or connected to RSS feeds for real-time monitoring.
❓ How confident should we be in the intelligence?
That's why we built confidence calibration: High = verified against official sources. Medium = single reliable source. Low = inference. Your team knows exactly what weight to give each item.
❓ Can signal priorities be customized?
Absolutely. If certifications matter more to Wilkins than Drains, we can tune that per-BU. The prioritization is fully configurable.
