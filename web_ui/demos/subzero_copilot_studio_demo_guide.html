1

Create HR Self-Service Agent in M365 Copilot

Microsoft 365 Copilot  ·  Agents Tab
~3 min
  • 1
    Open M365 Copilot

    Navigate to Microsoft 365 Copilot in your browser or switch to the pre-loaded tab.

    https://m365.cloud.microsoft/copilot
  • 2
    Click "Agents" in the right sidebar

    In the Copilot chat interface, find the Agents icon or panel on the right side.

  • 3
    Click "+ New Agent"

    This opens the M365 Copilot agent creation wizard — a lightweight, no-code experience.

  • 4
    Describe the agent to Copilot

    In the chat prompt, type a description like:

    "Create an HR Self-Service agent that helps Sub-Zero employees find HR policies, check their PTO balance, and get answers about benefits."
  • 5
    Review the auto-generated agent config

    Copilot will suggest a name, icon, description, and starter questions. Point out how fast this is — from idea to draft agent in seconds.

💬 Talking Point
"What you're seeing is the power of Microsoft 365 Copilot — any employee or IT admin can spin up an agent in minutes, no development skills required. Sub-Zero employees could have a personal HR assistant that knows your exact policies, not generic internet answers."
✅ The M365 Copilot wizard is great for the "wow" moment — it shows non-technical executives how approachable agent creation is before you dive into Studio.
2

Open in Copilot Studio

Copilot Studio  ·  Agent Editor
~1 min
  • 1
    Click "Open in Copilot Studio"

    In the M365 Copilot agent wizard, find the "Open in Copilot Studio" button (typically in the top-right or bottom of the panel). Click it.

  • 2
    Agent opens in Copilot Studio editor

    The agent you just created is now in Copilot Studio — fully editable. The settings, topics, and knowledge from M365 Copilot carry over automatically.

    https://copilotstudio.microsoft.com
  • 3
    Pause and orient the audience

    Point out the key sections of the Studio UI: Overview, Topics, Knowledge, Actions, and Publish.

💬 Talking Point
"Copilot Studio is where the real customization happens. Think of M365 Copilot as the fast-launch experience — and Copilot Studio as the full control panel. One click takes you from 'good enough' to 'enterprise ready.' And everything stays connected."
✅ If the browser pop-up is blocked, open Copilot Studio in the pre-loaded tab and navigate to the agent manually via Agents list.

Customize Agent — SharePoint Knowledge & Settings

Copilot Studio  ·  Knowledge & Topics
~5 min
Knowledge Source — SharePoint
  • 1
    Go to "Knowledge" tab → "+ Add knowledge"

    In the left nav, click Knowledge, then + Add knowledge source.

  • 2
    Choose "SharePoint" as the source

    Select SharePoint from the knowledge source options.

  • 3
    Paste the SharePoint library URL

    Enter the HR Policy library URL and click Add:

    https://d365demotsce30330346.sharepoint.com/sites/HumanResources/
    HR Policy and Benefit Documentation
  • 4
    Wait for indexing confirmation

    Copilot Studio will confirm the source is connected. The 7 HR documents (Employee Handbook, PTO Policy, Benefits Guide, etc.) are now the agent's knowledge base.

4

Create a New Agent from Scratch — Sub-Zero.com

Copilot Studio  ·  Website Knowledge Source
~4 min
  • 1
    Go back to Copilot Studio home → "+ New agent"

    Click the Agents breadcrumb or home icon, then click + New agent to start fresh.

  • 2
    Describe the agent in the wizard

    In the description box, type:

    "A customer assistant for Sub-Zero Wolf that helps customers explore products, find specifications, and locate dealers."
  • 3
    Add www.subzero.com as a knowledge source

    Under Knowledge, click + Add, choose Public website, and enter:

    https://www.subzero.com
  • 4
    Click "Create" and let Studio generate the agent

    Studio will scaffold the agent with topics, conversation starters, and the website knowledge source automatically.

  • 5
    Test with product-focused questions

    Open the test panel and try:

    "What refrigerator models does Sub-Zero make?"
    "What's the difference between the Designer and Classic series?"
    "Where can I find a Sub-Zero dealer near me?"
  • 6
    Show publish options (don't publish — just show)

    Click Publish in the nav to show channels: Teams, Website (embed), M365 Copilot, Dynamics 365. Point out how one agent deploys everywhere.

💬 Talking Point
"We just built a Sub-Zero product expert in under 5 minutes. Your website is already the knowledge base — Copilot Studio reads it, understands it, and turns it into a conversational experience. You could embed this agent on subzero.com, put it in Teams for your sales team, or make it part of your Dynamics 365 customer service — all from one place, with one click."
✅ Closing line: "Two agents. Two use cases. Zero code. That's what Microsoft Copilot Studio delivers for Sub-Zero."
